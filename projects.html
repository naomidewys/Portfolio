Vanilla Weather Project

In the SheCodes Plus workshop, I had to create a Vanilla weather app using HTML, CSS and Javascript. It fetches live data from the SheCodes API and displays the current temperature in celsius or fahrenheit. The current day's high/low range and wind speed and the 5-day forecast are shown in metric units.

React Weather Project

In the SheCodes React workshop, I had to create a weather app using HTML, CSS, JavaScript and React.js. It fetches live data from the SheCodes API and displays the current temperature in celsius or fahrenheit. The current day's wind speed and the 5-day forecast are shown in metric units. The weather icons use an animated react template.
